Display

Display .display classes gives the user to control the content visibility across the breakpoints.

I'm visible only on small devices.

I'm visible only on medium devices.

I'm visible only on large devices.

Sample Code:


<div class="display-tb-down">
    <p class="alert alert-red">I'm visible only on small devices.</p>
</div>
<div class="display-tb-to-lp">
    <p class="alert alert-blue">I'm visible only on medium devices.</p>
</div>
<div class="display-lp-up">
    <p class="alert alert-green">I'm visible only on large devices.</p>
</div>
